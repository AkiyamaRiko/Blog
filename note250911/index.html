    数组索引

    • indexOf(); 在数组中查找给定元素的第一个索引

    如果存在则返回索引号，如果不存在则返回-1

    • lastIndexOf(); 在数组中的最后一个索引（从后往前查找）

    如果存在则返回索引号，如果不存在则返回-1

    案例：

    // 数组去重
// 原理：遍历旧数组，用旧数组元素查询新数组
// 用indexOf判断元素在新数组中是否存在

function unique(arr) {
    const newArr = [];
    for (let i = 0; i < arr.length; i++){
        if (newArr.indexOf(arr[i]) === -1) {
            newArr.push(arr[i]);
        }
    }
    return newArr;
}

    字符串对象

    • 基本包装类型：包括String、Number、Boolean三种，是把简单数据类型包装成为复杂数据类型，使其有属性和方法。

    • 字符串所有的方法都不会修改字符串本身，操作完成后会返回新的字符串，旧的也没有消失。

    • indexOf(); 也可以用于字符串。

    • indexOf('要查找的字符串',起始查找位置); 可以从指定位置开始查找。

    // 查找字符串"arcaea"中所有的a出现的位置和次数

const str = 'arcaea'
let index = str.indexOf('a');
let num = 0;
while(index !== -1){
    console.log(index);
    index = str.indexOf('a', index + 1);
    num++;
}
console.log('a出现的次数是' + num);

    根据位置返回字符

    • charAt(index) 根据索引号返回字符

    • charCodeAt(index) 根据索引号返回字符的ASCII码

    • str[index] 获取指定位置的字符

    // 案例：判断一个字符串里出现次数最多的字符
// 并统计次数
// 1.用charAt遍历字符串
// 2.把每个字符存储给对象
// 3.遍历对象
const str1 = 'arcaea';
const o = {};
for (let i = 0; i < str1.length; i++){
    const chars = str1.charAt(i);
    if (o[chars]) {
        o[chars]++;
    } else {
        o[chars] = 1;
    }
}
console.log(o);

let max = 0
let ch = ''
for (var k in o) {
    if (o[k] > max) {
        max = o[k];
        ch = k;
    }
}
console.log(ch + ',' + max);

    字符串操作方法

    • concat('字符串1','字符串2'……)拼接字符串，等效于+

    • substr(start,length) 从start（索引号）位置开始，length是取出的个数。

    • slice(start,end) 从start开始截取到end（不包括end）

    • substring(start,end) 从start开始截取到end（不包括end）但是不接受负值。

    • replace('a','b') 替换字符，把a替换成b

    • split('分隔符') 把字符串转换为数组

    • toUpperCase() toLowerCase 转换大小写

    数据类型

    简单类型与复杂类型

    • 简单类型（值类型）：在存储时存储的是值本身，例如string、number、boolean、undefined

    • 复杂类型（引用类型）：在存储时只是存储了地址；可以通过new关键字新建，例如object、array、date

    堆和栈（数据结构中的概念）

    • 栈：由操作系统自动分配释放存放函数的参数值、局部变量的值等。简单数据类型存放到栈中。

    • 堆：存储复杂数据类型，由程序员分配释放，若程序员不释放，则由垃圾回收机制回收。复杂数据类型存放到堆中。

    • 复杂类型变量的栈空间存放的是地址，而真正的对象存放在堆空间中。
