    while语句

    语法结构：

    while (条件表达式){
    // 循环体
}

    执行思路：当条件表达式结果为true时，则执行循环体，否则退出循环。

    var num = 1;
while (num <= 100){
    console.log('hello');
    num++;
}

    和for循环语句一样需要有计数器和操作表达式，防止变成无限循环把电脑卡死。

    do while 语句

    语法结构：

    do {
    // 循环体
} while (条件表达式)

    执行思路：与while语句不同之处在于：先执行一遍再判断条件。

    (也就是说，无论如何都会至少执行一次循环体代码。)

    var i = 1;
do {
    console.log('hello');
    i++;
} while (i <= 100)

    continue 和 break

    • continue 用于立即跳出本次循环，进入下一次循环。
    • break用于让循环直接结束。
