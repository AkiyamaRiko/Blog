    • select 下拉表单
    <select>
    <option>选项1</option>
    <option>选项2</option>
    <option selected="selected">……</option>
</select>

    在option中定义selected=“selected”，表示默认选中。

    • textarea 用于定义多行文本
    <textarea rows="3" cols="20">
    文本内容……
</textarea>

    cols="每行的字符数" rows="行数"，实际开发中不常用。

    CSS

    HTML的局限性

    html只关注语义、内容，所以网页美化、布局主要靠CSS。

    CSS语法规范

    • CSS语句的构成：选择器{声明;}
    • 选择器用于指定html标签，而声明决定了具体样式。
    • 声明是属性与属性值构成的键值对。
    • 属性与属性值之间用分号隔开。
    • 便于开发的代码风格
    • 样式：建议使用展开式
    • 不要用大写
    • 属性值前面和冒号后面留一个空格、选择器与大括号之间保留一个空格。
    h3 {
    color= blue； 
}

    选择器

    基础选择器：标签、类、id、通配符选择器

    • 标签选择器直接以标签名为选择器，可以把某一类标签全都选出来，可以快速把页面上的同一标签设置为统一样式。
    • 类选择器需要调用class属性。例如：

    在html中

    <div class="arcaea"> 这是文本 </div>
<!--要先在html中定义类名-->

    在对应的CSS中

    .arcaea {
    color: #1f1e33; 
}

    尽量用纯英文类名。 一个标签可以有多个类名，用空格隔开。选择其中一个类名就能选中这个标签，这个标签可以分别具有这些类的样式。

    .arcaea {
    width: 100px;
    height: 100px; 
}
.empurple {
    background-color: #1f1e33;
}

    <div class="arcaea empurple">

    这两段代码合起来会得到一个100*100的紫色矩形。

    • ID选择器：调用ID属性，每个html中只能出现一次
    .arcaea {
    width: 100px;
    height: 100px; 
}
#empurple {
    background-color: #1f1e33;
}

    <div class="arcaea" id="empurple">

    可以把一些标签元素相同的样式（共同的部分）放进一个类里，可以节省代码、便于修改。

    • 通配符选择器：*{属性}，选取所有元素，一般用于清楚元素的内外边距
    * {
    margin: 0;
    padding: 0; 
}

    字体

    font-family

    • 各种字体之间用逗号隔开
    • 由多个单词组成的字体名要加上引号
    • 最好是把中英文的都选上，兼容性会比较好。电脑会自动选取排在前面的。

    font-size

    • 单位为px（不要忘了写单位）
    • 谷歌浏览器默认的文字大小是16px，但各种浏览器默认的文字大小不一样，最好在body中把整个页面的文字指定为16px（标题需要另外指定）

    font-weight

    • 包括normal、bold、bolder、lighter
    • 也可以用数字（100-900），没有单位。
    • 700等价于bold，400等价于normal

    font-style

    • normal或italic（斜体）
    p {
    font-family: Arial,"Microsoft YaHei","微软雅黑"; 
    font-size: 20px; 
    font-weight: 700;
    font-style: normal;
}

    也可以用复合的写法：(顺序是固定的，弄反了就不生效了)

    p {
    font:italic 700 16px/20px "Microsoft YaHei"; 
    /*style weight font-size/line height family*/
}

    文本属性

    颜色

    • color ；预定义的颜色/十六进制代码/rgb值
    • red、blue、green……
    • 1f1e33
    • rgb(255,0,0)

    对齐

    • text-align用于设置文本的水平对齐方式
    • 属性值：left、right、center

    装饰文本

    • text-decoration，用于添加上划线、下划线、删除线
    • 属性值：overline、underline、line-through、none（什么也没有）

    缩进

    • text-indent用于设置段落首行缩进，可以以px为单位。
    • 相对单位em：当前元素中一个文字的大小。
    • 如果需要首行缩进两个字的距离，可以把text-indent设置为2em

    行高

    • line-height用于设置行高。单位可以是px或em

    CSS的引入方式

    内部样式表

    把CSS样式写在html文件的style标签内部

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Welcome to Arcaea</title>
    <style>
        /*!这里面写样式*/
    </style>
</head>

    行内样式表

    把CSS样式直接写在标签里，适合用于修改单个元素的样式，但是一般来说不建议

    <div style="color: red;"></div>

    外部样式表

    样式单独写在CSS文件里，然后再把CSS引入到html中。（代码很多的时候推荐）

    <link rel="stylesheet" href="style.css">
<!--写在head里-->

    调试工具

    浏览器按F12，可以随便改

    Emmet语法

    使用缩写，写完缩写以后按下tab，自动生成对应的代码，提高html/css的编写速度。VScode已经集成该语法。

    • 快速生成html
    • 输入多个相同标签：例如div*10 可以直接生成10个div
    • 输入父子关系的标签，用> :例如ul>li
    • 输入兄弟关系的标签，用+ :例如div+p
    • 输入带有类名的标签用. ,带id的用#
    • 如果需要排序，用$
    • 符号两边不需要空格
    • 快速生成CSS（建议熟练了再用）

    复合选择器

    由基本选择器组合而成，可以更准确、更高效地选择元素，包括后代选择器、子选择器、并集选择器、伪类选择器等。

    后代选择器

    在想要选中的父子元素之间加空格 元素1 元素2 {样式声明;} 假设有这样一段html代码：

    <ol>
    <li>我是ol</li>
    <li>我是ol</li>
<ol>
<ul>
    <li>我是ul</li>
    <li>我是ul</li>
<ul>

    这时如果想要选中ol里的li：

    ol li {
    color: #1f1e33
}

    如果元素1里有元素2，而元素2里还有元素3，也可以这样选中。

    元素1 元素2 元素3 …… {样式声明;}

    这些元素可以是任意基础选择器

    子选择器

    子选择器只能选择最近的一级子集元素 简单来说就是只选亲儿子 元素1>元素2 {样式声明;}

    例1

    <div class="nav">
    <ol>
        <li><a href="#">1f1e33</a></li>
    <ol>
</div>

    如果想选中目标文字，可以这样写：

    .nav ol li a {
    color: #1f1e33;
}

    例2

    <div class="nav">
    <a href="#">1f1e33</a>
    <ol>
        <li><a href="#">not 1f1e33/a></li>
    <ol>
</div>

    如果想选中目标文字，可以这样写：

    .nav>a {
    color: #1f1e33;
}
