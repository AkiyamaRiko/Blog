CommentsTool/抖音评论整理工具

2025.8 萌新的第一篇技术笔记

前言（碎碎念）

这是萌新学会Python以后写的第一个正儿八经的程序。这篇文档也算是Markdown练习吧。在公司每天午睡起来以后都要在Excel上整理抖音直播间评论，包括评论用户的昵称、ID、IP属地、电话号码……，很机械很无聊，时间长了以后每次看到Excel都觉得烦了，于是开发了这个。其实最后文件还是要在excel打开的，这个工具的实际作用不大，就当是个练习吧。（本来是想学一下爬虫的，但是有点怕造成法律问题，而且抖音也有反爬机制，算了算了）

界面与功能描述

窗口中从上到下5个输入框，依此是评论者的昵称、id、ip属地、电话号码、评论内容。最下面是保存按钮，点一次保存一条记录。enter键也可以保存。Ctrl+E则是导出所有评论为csv文件，然后可以在Excel打开。

正文部分

做这种简单的基于窗口的程序可以用Tkinter＋Pandas。

import tkinter as tk
from tkinter import messagebox # messagebox是对话框的意思
import pandas as pd
from datetime import datetime  #准备用当前时间命名导出的文件
import tkinter.font as tkFont  #用来把字体改大
from ctypes import windll      #调用Windows API，用来做高分屏适配

然后先初始化。

 def __init__(self, root):
        self.root = root
        self.data = []
        self.setup_ui()
        self.setup_dpi() # 准备做高分屏适配

然后开始字段定义

    def setup_ui(self):      
        self.fields = ["昵称", "用户ID", "IP属地", "电话", "评论内容"]
        self.entries = {}
        default_font = tkFont.Font(family="微软雅黑", size=13)
        self.root.option_add("*Font", default_font) # 改字体
        for i, field in enumerate(self.fields):
            label_style = {"bg": "white", "fg": "black"}  #改底色（默认灰色太丑了）
            tk.Label(self.root, text=field,**label_style).grid(row=i, column=0, padx=5, pady=5)
            entry = tk.Entry(self.root, width=40)
            entry.grid(row=i, column=1, padx=5, pady=5)
            self.entries[field] = entry
