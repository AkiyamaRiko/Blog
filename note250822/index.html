    自定义列表

    前面没有项目符号，也没有数字编号。

    <dl>
    <dt>在这里输入</dt>
    <dd>列表内容1</dd>
    <dd>列表内容2</dd>
</dl>

    dl里面只能放dt和dd。dl和dd里面可以放各种元素。

    表单

    表单用于收集用户的信息。 一个完整的表单通常由表单域、表单控件、提示信息组成。

    • form 表单域 用于确定表单的范围，它决定了要把哪些信息提交给服务器。
    • action=“url地址” 用于指定接受数据的服务器地址。
    • method=“提交方式” 分为get/post。
    • name=“名称”
    • 表单控件
    • input 输入
      • type=“button / checkbox / file / hidden / image / password / radio(单选) / reset / submit / text……” 决定了输入的类型
      • name=“名称”，一组单选按钮必须得用相同的name
      • value=“默认显示文字”，
      • checked=“checked” 打开时默认选中
      • maxlength=“最大字符长度”，正整数
    • select 下拉选择
    • textarea 文本域
    • 提示信息
    • label 常用于绑定表单元素：当用户点击label标签的内容时，会自动选中对应的表单元素。
    <label for="username">用户名</label><input type="text" id="username">
