    HTML¶

    HTML基本标签和语法¶

    • 大部分标签都是双标签
    • 最基本的骨架标签:
    • html
    • head（包括编码信息、内部样式表等）
      • title（网页的标题）
    • body（网页的主要内容）

    在vscode里写html，第一件事是输入一个感叹号，按tab，会自动生成骨架。

    常用标签小词典¶

    <h1>这是标题</h1>
<h6>标题最多6级</h6>
<p>这是段落</p>
<p>强制<br />换行</p>

    表格标签¶

    不用来布局，只用来展示数据。

    <table> <!--定义表格-->
  <thead> <!--表头语义标签-->
  <tr>  <!--定义一行--> 
    <th>表头</th> <!--会自动加粗居中-->
  </tr>
  </thead>  
  <tbody><!--表格主体语义标签-->
  <tr>
    <td>单元格</td><!--定义一格-->   
  </tr>
  </tbody>
<table>
    • thead标签内必须得有第一行的tr标签。
    • 表格属性包括align（对齐方式）、border（边框）、cellpadding（单元格内空白）、cellspacing（单元格间空白）、width（宽度）。这些属性会在CSS中用到。
    • 合并单元格属性：
    • rowspan：跨行合并
    • colspan：跨列合并
    • 这两个属性代码写在要合并的单元格中最左、最上的单元格里，写上要合并的数量。
    • 例如：
    <td colspan="2"></td>

    合并完记得把多余的单元格删掉。

    自定义列表¶

    前面没有项目符号，也没有数字编号。

    <dl>
    <dt>在这里输入</dt>
    <dd>列表内容1</dd>
    <dd>列表内容2</dd>
</dl>

    dl里面只能放dt和dd。dl和dd里面可以放各种元素。

    表单¶

    表单用于收集用户的信息。 一个完整的表单通常由表单域、表单控件、提示信息组成。

    • form 表单域 用于确定表单的范围，它决定了要把哪些信息提交给服务器。
    • action=“url地址” 用于指定接受数据的服务器地址。
    • method=“提交方式” 分为get/post。
    • name=“名称”
    • 表单控件
    • input 输入
      • type=“button / checkbox / file / hidden / image / password / radio(单选) / reset / submit / text……” 决定了输入的类型
      • name=“名称”，一组单选按钮必须得用相同的name
      • value=“默认显示文字”，
      • checked=“checked” 打开时默认选中
      • maxlength=“最大字符长度”，正整数
    • select 下拉选择
    • textarea 文本域
    • 提示信息
    • label 常用于绑定表单元素：当用户点击label标签的内容时，会自动选中对应的表单元素。
    <label for="username">用户名</label><input type="text" id="username">
    • select 下拉表单
    <select>
    <option>选项1</option>
    <option>选项2</option>
    <option selected="selected">……</option>
</select>

    在option中定义selected=“selected”，表示默认选中。

    • textarea 用于定义多行文本
    <textarea rows="3" cols="20">
    文本内容……
</textarea>

    cols="每行的字符数" rows="行数"，实际开发中不常用。
