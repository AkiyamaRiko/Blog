    内置对象

    • 内置对象是JS自带的一些对象，提供了很多基本的、必要的功能（属性和方法）

    • 常用的有Math、Date、Array、String……

    • 通过查文档学习内置对象：MDN Web Docs

    • 如何学习对象中的方法：查阅方法的功能 → 查看参数的类型和意义 → 查看返回值的类型和意义 → 通过demo进行测试

    Math对象

    • 使用Math对象不需要new，直接用里面的属性和方法即可。

    • 属性例子：圆周率 Math.PI

    • 最大值：Math.max( ); 最小值：Math.min( ); 括号内输入一组数值。

    #### 案例：封装自己的数学对象

    js const myMath = { // 里面封装了一个属性、两个方法 PI: 3.1415926, max: function () { let max = arguments[0]; for (let i = 1; i < arguments.length; i++){ if (arguments[i] > max) { max = arguments[i]; } } return max; }, min: function() { let min = arguments[0]; for (let i = 1; i < arguments.length; i++){ if (arguments[i] < min) { min = arguments[i]; } } return min; }, }

    #### Math对象中其他常用的方法：

```js
Math.floor(); // 向下取整
Math.ceil(); // 向上取整
Math.round(); // 四舍五入
Math.abs(); // 绝对值，如果输入了字符串，会进行隐式转换
Math.random(); // 随机返回一个0~1之间的小数

    关于random：

    • 括号里不需要填参数

    • 取值范围是[0,1)

    • 如果需要得到两个数之间的随机整数（并且包含这两个整数）

    Math.floor(Math.random() * (max - min + 1)) + min;

    • 这个很有用，可以封装起来

    Date对象

    • 使用Date对象需要用new

    • 参数常用的写法：数字型或字符串型

    const date = new Date();
console.log(date); // 2025-09-10T06:33:43.544Z

    格式化日期

    通过以下方法可以单独获取日期指定的部分。

    • getFullYear(); 获取当年

    • getMonth();获取当月（0~11，如果需要真实的月份，要在后面+1）

    • getDate();获取当天日期

    • getDay();获取星期几（0~6，周日到周六）

    • getHours();获取小时，getMinutes();获取分钟，getSeconds();获取秒钟。

    • 输出时可以这样写：

    const y = date.getFullYear();
const m = date.getMonth() + 1;
const d = date.getDate();
console.log('今天是' + y + '年' + m + '月' + d + '日');
    • getTime(); 或者 valueOf(); 或者Date.now();获取从1970.1.1到现在的毫秒数（时间戳）

    倒计时

    • 思路：输入的时间减去现在的时间就是倒计时

    • 用时间戳来做，得到毫秒数倒计时

    • 把毫秒转换为天数、时、分、秒

    function countDown(time) {
    const now = +new Date();
    const input = +new Date(time);
    const times = (input - now) / 1000;
    const d = parseInt(times / 60 / 60 / 24);
    const h = parseInt(times / 60 / 60 % 24);
    const m = parseInt(times / 60 % 60);
    const s = parseInt(times % 60);
    return d + '天' + h + '小时' + m + '分钟' + s + '秒';
}
console.log(countDown('2028-3-1 18:00'));

    数组对象

    • 创建数组的两种方式 const arr = []; 或者 const arr = new Array();

    • instanceof运算符，可以用来检测是否为数组：

    console.log(arr instanceof Array); 返回true或false

    • Array.isArray(); 可以用来检测括号里的内容是否为数组：

    console.log(Array.isArray(arr)); 返回true或false

    添加或删除数组元素

    1. push() 在数组的末尾增加元素

    arr.push(3); 添加一个

    arr.push(4, 5, 6, 7); 添加多个

    如果console.log(arr.push(3));这样操作，返回新数组的长度

    1. upshift() 在数组的开头增加元素

    arr.upshift(3); 添加一个

    arr.upshift(4, 5, 6, 7); 添加多个

    如果console.log(arr.upshift(3));这样操作，返回新数组的长度

    1. pop() 删除数组的最后一个元素

    arr.pop();

    不用填参数，执行过后返回的是删掉的那个元素

    1. shift() 删除数组的第一个元素

    arr.shift();

    不用填参数，执行过后返回的是删掉的那个元素

    数组排序

    1. 翻转数组： arr.reverse();

    2. 排序：arr.sort();

    如果只用sort排序，而数组中有各种不同位数的数值，则无法准确做到升序或降序，但在sort()中插入function，可以控制升序/降序。

    ```js const arr1 = [1, 2, 3]; arr1.reverse(); // 翻转数组 console.log(arr1);

    const arr2 = [1, 34, 2, 8, 67, 25]; arr2.sort(function (a, b) { return a - b;// 升序 return b - a;// 降序 }) ```
