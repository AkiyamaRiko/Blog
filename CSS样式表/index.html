    CSS¶

    教程有点过时，仅供参考。

    HTML的局限性¶

    html只关注语义、内容，所以网页美化、布局主要靠CSS。

    CSS语法规范¶

    • CSS语句的构成：选择器{声明;}
    • 选择器用于指定html标签，而声明决定了具体样式。
    • 声明是属性与属性值构成的键值对。
    • 属性与属性值之间用分号隔开。
    • 便于开发的代码风格
    • 样式：建议使用展开式。
    • 不要用大写。
    • 属性值前面和冒号后面留一个空格、选择器与大括号之间保留一个空格。
    h3 {
    color= blue； 
}

    基础选择器¶

    基础选择器：标签、类、id、通配符选择器

    • 标签选择器直接以标签名为选择器，可以把某一类标签全都选出来，可以快速把页面上的同一标签设置为统一样式。
    • 类选择器需要调用class属性。例如：

    在html中

    <div class="arcaea"> 这是文本 </div>
<!--要先在html中定义类名-->

    在对应的CSS中

    .arcaea {
    color: #1f1e33; 
}

    尽量用纯英文类名。 一个标签可以有多个类名，用空格隔开。选择其中一个类名就能选中这个标签，这个标签可以分别具有这些类的样式。

    .arcaea {
    width: 100px;
    height: 100px; 
}
.empurple {
    background-color: #1f1e33;
}

    <div class="arcaea empurple">

    这两段代码合起来会得到一个100*100的紫色矩形。

    • ID选择器：调用ID属性，一个ID在每个html中只能出现一次**

    文本属性¶

    字体属性¶

    font-family¶

    • 各种字体之间用逗号隔开
    • 由多个单词组成的字体名要加上引号
    • 最好是把中英文的都选上，兼容性会比较好。电脑会自动选取排在前面的。

    font-size¶

    • 单位为px（不要忘了写单位）
    • 谷歌浏览器默认的文字大小是16px，但各种浏览器默认的文字大小不一样，最好在body中把整个页面的文字指定为16px（标题需要另外指定）

    font-weight¶

    • 包括normal、bold、bolder、lighter
    • 也可以用数字（100-900），没有单位。
    • 700等价于bold，400等价于normal

    font-style¶

    • normal或italic（斜体）
    p {
    font-family: Arial,"Microsoft YaHei","微软雅黑"; 
    font-size: 20px; 
    font-weight: 700;
    font-style: normal;
}

    也可以用复合的写法：(顺序是固定的，弄反了就不生效了)

    p {
    font:italic 700 16px/20px "Microsoft YaHei"; 
    /*style weight font-size/line height family*/
}

    颜色¶

    • color ；预定义的颜色/十六进制代码/rgb值
    • red、blue、green……
    • 1f1e33
    • rgb(255,0,0)

    对齐¶

    • text-align用于设置文本的水平对齐方式
    • 属性值：left、right、center

    装饰文本¶

    • text-decoration，用于添加上划线、下划线、删除线
    • 属性值：overline、underline、line-through、none（什么也没有）

    缩进¶

    • text-indent用于设置段落首行缩进，可以以px为单位。
    • 相对单位em：当前元素中一个文字的大小。
    • 如果需要首行缩进两个字的距离，可以把text-indent设置为2em

    行高¶

    • line-height用于设置行高。单位可以是px或em

    背景属性¶

    • background-color 定义了元素的背景颜色，默认是transparent。
    • background-image 定义了元素的背景图片，实际开发常见于logo或者装饰性的小图片。url（路径）
    • background-repeat 定义了背景图片平铺的方式：no-repeat/repeat/repeat-x（横向）/repeat-y（纵向）
    • 背景图片会压住背景颜色。
    • background-position：x y; 可以改变图片在背景中的位置，参数可以是方位或者精确数字。
    • 方位包括top、bottom、left、right、center。
    • 精确数字先写x再写y，单位为px。
    • 如果只指定其中一个数值，该数值一定是x坐标，则另一个默认居中。
    • background-attachment：scroll/fixed 设置背景图像固定，可以用来做视差效果。
    • 复合写法：按照颜色、图片路径、平铺方式、滚动、位置的顺序。
    • 半透明：
      background:rgba(0,0,0,0.5)

    最后一个是alpha(透明度)。

    CSS的引入方式¶

    内部样式表¶

    把CSS样式写在html文件的style标签内部

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Welcome to Arcaea</title>
    <style>
        /*!这里面写样式*/
    </style>
</head>

    行内样式表¶

    把CSS样式直接写在标签里，适合用于修改单个元素的样式，但是一般来说不建议

    <div style="color: red;"></div>

    外部样式表¶

    样式单独写在CSS文件里，然后再把CSS引入到html中。（代码很多的时候推荐）

    <link rel="stylesheet" href="style.css">
<!--写在head里-->

    复合选择器¶

    并集选择器¶

    用于选择多组标签，同时为他们定义相同的样式。 元素1, 元素2, 元素3……{样式声明} 按照约定的语法规范，并集选择器要竖着写。最后一个选择器后面不需要逗号。

    伪类选择器¶

    用于向某些选择器添加特殊效果，比如给链接添加特殊效果，或选择第一个、第n个元素，用冒号表示，例如:hover,:first-child

    链接伪类选择器：

    • a:link 选择所有未被访问的链接
    • a:visited 选择所有已被访问的链接
    • a:hover 选择鼠标指针位于其上的链接
    • a:active 选择鼠标按下未弹起的链接
    • 按照LVHA的顺序声明。

    :focus伪类选择器： 用于选取焦点（光标所在位置）的表单元素，最常用的是input:focus这个写法。

    CSS的元素显示模式¶

    html中的元素分为块元素和行内元素。

    块元素¶

    • 常见的块元素有h1~h6、p、div、ul、ol、li,其中最典型的是div。
    • 高度、宽度、内外边距都可以控制。
    • 宽度默认是容器的100%，可以独占一行。
    • 是容器/盒子，里面可以放行内元素或者块元素。(h和p里面只能放普通文字)

    行内元素/内联元素¶

    • 相邻行内元素在一行上，一行可以显示多个。
    • 高、宽直接设置是无效的。
    • 默认宽度是本身内容的宽度。
    • 行内元素只能容纳文本和其他行内元素。
    • 链接里面不能再放链接，但链接里可以放块元素。

    行内块元素¶

    特殊标签img/、input/、td，同时具有块元素和行内元素的特点。

    • 和相邻行内元素在同一行上，但是他们之间有空白间隙。一行可以显示多个。
    • 默认的宽度就是它本身内容的宽度。
    • 高度、行高、内外边距可以控制。

    元素显示模式的转换¶

    • display: block 转换为块元素，可以设置宽度和高度了。
    • display: inline 转换为行内元素，此时不再能设置宽度和高度，但是一行可以放置多个。
    • display: inline-block 转换为行内块。

    CSS的特性¶

    • 层叠性：样式冲突时，遵循就近原则：哪个样式离结构近，就遵循哪个，冲突的会被覆盖，不冲突的正常执行。
    • 继承性：子标签会继承父标签的某些样式（按f12调试时，会看到inherited）（主要是文字样式）
    • tips:如果用em定义了父元素的行高，子元素也可以继承，如果两个元素的字体大小设置得不一样，子元素的实际行高会是子元素的字体大小*继承到的em倍数。
    • 优先级：如果给同一个元素指定了多个选择器：
    • 若选择器相同，执行层叠性。
    • 若选择器不同，根据权重执行：！important>行内样式>id选择器>类选择器>元素选择器>继承或通配符
    • 使用复合选择器时，会出现权重叠加，这时候要比较总权重。

    盒子模型¶

    网页元素基本上都是“盒子”。布局网页时，先利用CSS设置好盒子的样式，然后摆放到相应位置，再往盒子里面加内容。 盒子包括边框、内边距、外边距、内容。

    边框：有width、style、color三种属性，可以分开写也可以复合写（没有顺序）

    • border-collapse 控制相邻表格边框是否合并。
    • 边框会影响盒子的实际大小。

    内边距 padding：内容到边框的距离，上下左右可以分开设置，也可以复合写：

    • 如果只写一个值，表示上下左右
    • 写2个值，分别表示上下和左右
    • 写3个值，表示上、左右、下
    • 写4个值，表示上、右、下、左

    外边距 margin：和padding一样可以分开设置也可以复合写。

    页面布局的思路¶

    • 确定版心：一般是1200px宽，居中。可以设置一个类，专门用来做版心。
    • 分析页面的行模块、以及每行的列模块。
    • 制作HTML的结构。
    • CSS书写顺序：布局与定位、自身属性（长宽高、内外边距）、文本属性、其他属性。

    布局方法¶

    (让AI帮忙整理)

    /* 基础样式（移动端） */
.container { 
  display: flex; 
  flex-direction: column; 
}

/* 中等屏幕 */
@media (min-width: 768px) {
  .container { flex-direction: row; }
}

/* 大屏幕 */
@media (min-width: 1024px) {
  .container { 
    display: grid;
    grid-template-columns: 1fr 3fr;
  }
}

    制作一个复杂的页面的时候，可以给页面的各个部分分别创建Container，然后根据实际情况使用不一样的布局方式。例如：

    • Header和导航栏使用Flex，水平排列。

    • 主体内容用grid。

    • 页脚用flex。

    （AI给的例子）

    er {
    grid-template-columns: repeat(2, 1fr); /* 平板端：2列 */
  }
}

/* 桌面尺寸 */
@media (min-width: 1024px) {
  .main-container {
    grid-template-columns: repeat(3, 1fr); /* 桌面端：3列 */
  }

  .site-layout {
    display: grid;
    grid-template-columns: 250px 1fr; /* 侧边栏+主内容 */
  }
}
