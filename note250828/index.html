    输入输出语句

    alert(msg);
//浏览器弹出警示框
console.log(msg);
//浏览器控制台打印输出信息
prompt(info);
//浏览器弹出输入框，让用户输入

    变量

    变量是存放数据的容器。

    声明和赋值

    var x; //声明一个名称为x的变量
x = 10 ;//赋值
var x = 10,y = 'DABCABC' ;//也可以声明变量的同时直接赋值,用逗号隔开
//变量可以是字符串（？
console.log(x);//在控制台输出（一般是测试用的，只有程序员能看见）

    变量的数据类型

    JS的数据变量类型是赋值的时候才确定的，并且类型可以变化。

    • 简单数据类型：Number（包括整数和小数）、Boolean（布尔型，如true、false）、String（字符串）、Undefined（未定义）、Null（空值）。

    • 复杂数字类型：Object等。

    数字型

    • 八进制
    var num = 010 ;//在数字前面加个0表示8进制
console.log(num) ;
//这时候会输出十进制结果，也就是8
    • 十六进制
    var num = 0x9 ;//数字前面加0x表示十六进制
//这时候运行console.log也会输出十进制结果
    • 数字型的范围：

    最大值是1.79e＋308，最小是5e-32

    • 三个特殊值：正负infinity和NaN（Not a number）

    判断一个值是否为数字：

    isNaN(x) ;//用来判断变量是不是数字，如果是，返回false
//如果不是，返回true
console.log(isNaN(x));

    字符串

    字符串需要写在引号内，用单引号或双引号都可以，在html中建议用单引号。

    var str1 = '我喜欢玩Arcaea';
var str2 = '我喜欢玩\n Arcaea';//用转义字符可以换行
    • 长度（length）
    var str = 'my name is riko';
console.log(str.length); //15
    • 拼接：字符串+另一个字符串/数字/布尔值，会得到一个拼接的字符串。
    var age = 18;
console.log('您今年' + age + '岁了');
//会得到“您今年18岁了”的字符串


var age = prompt('请输入您的年龄');
//此时会出现弹窗，让用户输入值
//然后会把用户输入的值赋值给变量

console.log('您今年' + age + '岁了');
//会得到“您今年__岁了”的字符串(取决于刚才输入的值)


alert('您今年' + age + '岁了');
//会出现相应的弹窗

    布尔值

    参与加法运算时，true视为1，false视为0

    var flag = true;
console.log(flag+1); //会得到2

    其他

    • 如果变量声明了但没有赋值：Undefined，但可以和字符串相加，得到拼接的字符串。

    • NaN、Undefined和数字相加，结果是NaN。

    • Null和字符串相加，得到拼接的字符串。

    • Null和数字相加，Null相当于0。

    判断变量的数据类型：typeof

    例：

    var num = 10;
console.log(typeof num);
// number
var str = '10';
console.log(typeof str);
// string
var timer = null
console.log(typeof timer);
// object

    • 通过表单、prompt取过来的值会变成字符型。

    • 可以通过控制台输出的字体颜色来判断（字符串会显示为黑色）

    变量转换类型

    转换为字符串：

    var num = 1;
alert(num.toString());
var num = 1;
alert(String(num));
// 转换成字符串的两种方式

var num = 1;
alert(num+'我是字符串');
// 和字符串拼接可以得到字符串

    转换为数字型：

    var num = prompt('请输入一个数');
console.log(num);
// 会得到字符串

var num = prompt('请输入一个数');
console.log(parseInt(num)); // I要大写
// 会得到整数，如果输入的是小数的话会取整
console.log(parseFloat(num)); 
// 会得到浮点数

console.log(parseInt(120px));
// 120

console.log(parseInt(rem120px));
// NaN，因为会先识别到rem
var str = '123'
console.log(Number(str));
// 数字型的123
