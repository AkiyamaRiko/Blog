    编译与解释

    • 计算机只能理解机器语言。程序语言翻译成机器语言的工具称为翻译器。

    • 编译与解释的区别在于翻译的时间点不同。

    • 编译器是在代码执行之前编译，生成中间代码文件。例如JAVA

    • 解释器是在运行时进行即时解释，并立即执行。例如JS

    标识符、关键字、保留字

    • 标识符：开发人员为变量、属性、函数、参数起的名字。标识符不能是关键字和保留字。

    • 关键字：在语法中已经有特殊含义的单词。

    • 保留字：预留的关键字。

    运算符

    用来实现赋值、比较、执行算术运算等功能的符号。

    算术运算符

    • 加减乘除

    • % 取余数（返回除法的余数）例如 9 % 2 = 1

    • 避开浮点数运算，会有误差

    表达式和返回值

    • 表达式是由数字、变量、运算符组成的式子。

    • 表达式最终都会有一个结果返回，称为返回值。

    递增和递减运算

    用于反复给数字变量+1或-1

    // 前置自增：先加1再返回值
var num = 10;
++ num; // 类似于num = num + 1
console.log(num); // 11

// 后置自增：先返回原值再加1
var num = 10;
num ++; // 类似于num = num + 1
console.log(num); // 10
console.log(num); // 11

    比较运算

    比较两个数据，然后返回布尔值。（使用==时会把字符串数据转换为数字型）

    逻辑运算符

    • && 与(and)

    • || 或(or)

    • ！非(not)

    • && 比 || 的权重高

    console.log(3 > 5 && 3 > 2); // false
colsole.log(3 > 5 && 3 > 2); // true
console.log(!true); // false
    • 逻辑中断（短路）：当有多个表达式参与逻辑运算时，如果左边的表达式可以确定结果，就不再运算右边。
    console.log(123 && 456); // 456
console.log(0 && 456); // 0
// 表达式1 && 表达式2
// 如果表达式1为真，则返回表达式2
// 如果表达式1为假，则返回表达式1
// 从左开始，如果是真，就继续往下运算直到最后一个，返回最后一个值
// 如果是假，就返回当前的值

    console.log(123 || 456); // 123
console.log(0 || 456); //456
// 表达式1 || 表达式2
// 如果表达式1为假，则返回表达式2
// 如果表达式1为真，则返回表达式1
// 从左开始，如果是假，就继续往下运算直到最后一个，返回最后一个值
// 如果是真，就返回当前的值

    赋值运算符

    • = 直接赋值

    • += -= 加/减一个数以后再赋值

    • *= /= %= 乘、除、取模以后再赋值

    var num = 10;
num += 2; // 相当于num = num + 2
console.log(num); // 12
